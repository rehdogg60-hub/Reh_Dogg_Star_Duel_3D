# Reh Dogg Star Duel 3D

Reh Dogg Games Plus presents a browser-based 3D arcade space dogfight built with Three.js.

## Play

Open `index.html` in a browser or upload this folder to GitHub Pages.

## Controls

- WASD or Arrow Keys: steer the fighter
- Mouse: aim/turn assist
- Left Click or Space: fire lasers
- Shift: boost
- Mobile: use the on-screen flight and action buttons

## Optional Audio

Place these files in the same folder as `index.html`:

- `background_music.mp3`
- `shoot.wav`
- `hit.wav`
- `explosion.wav`
- `boost.wav`
- `powerup.wav`
- `button_click.wav`
- `victory_stinger.wav`
- `game_over_stinger.wav`

If any file is missing, the game still runs silently.
